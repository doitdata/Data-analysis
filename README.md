## Data-analysis
데이터 분석이라는 거창한 이름을 쓰기에는 그렇지 않은 거 같은 느낌적인 느낌이지만 
데이터를 사용하긴 하니까 일단 데이터 분석이라고 붙여 보겠습니다. 

파이썬을 배우면서 웹 크롤링과 판다스를 이용한 데이터 분석에 흥미를 느꼈고, 재밌는거 계속 해보자라고 생각했습니다. 
참고 한 책에서 시작을 해서 영감을 받아서 다른 주제로 발전을 시켜보기도 했습니다.
